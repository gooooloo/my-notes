# 问题

如下代码里变量`a`的初始化是线程安全的吗？

``` C++
struct A{ A() {} };
void foo() { static A a; }
```



# 答案

`C++11` 保证它是线程安全的。但`C++11` 之前的没有保证。

`_ZGVZ3foovE1a(%rip)`是一个标志位，用于判断 `static A a`是否已经被初始化了。如果是，则跳过；如果否。则锁住临界区，再问一次标志位，如果已经初始化，则跳过；如果否，则初始化后解锁临界区。
