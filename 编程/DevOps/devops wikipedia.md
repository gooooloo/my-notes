# 总结

## 1. 字面意思

**DevOps**（**Dev**elopment和**Op**erations的组合词）



## 2. 目的

把开发和运维揉到一起。



## 3. 定位

1. 不同部门协作

2. 不同工种协作

   

## 4. 手段

1. 尽可能自动化

2. 尽可能集成各种内容 ---- 单一系统化

   1. 代码

   2. 编译系统

   3. 上线命令

   4. bug维护

   5. everything

   6. 缺失的 ---- wiki？

   7. 缺失的 ---- IM？
